# Bom (봄) — Learn Korean Sweetly 🌸

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen.svg)](https://bom-korean-bunny.lovable.app)

**Bom** (Korean for "Spring") is a modern, interactive web application designed to help users master Korean sentence structures. Unlike standard translators, Bom breaks down translated sentences to teach you the *why* and *how* behind Korean grammar, making it the perfect companion for language learners.

## ✨ Features

* **Deep Structural Breakdown:** Type an English sentence and get a natural Korean translation alongside a complete breakdown of the grammar.
* **Particle & Word Order Analysis:** Visualizes Korean's Subject-Object-Verb (SOV) structure and highlights essential markers (particles) so you understand exactly how the sentence is constructed.
* **User Authentication:** Secure sign-in system to track your progress and daily practice.
* **Responsive Design:** A beautifully styled, modern interface that works flawlessly across desktop and mobile devices.

## 🛠️ Tech Stack

This project is built with a highly performant, modern TypeScript ecosystem:

* **Frontend:** [React](https://react.dev/) + [Vite](https://vitejs.dev/)
* **Language:** [TypeScript](https://www.typescriptlang.org/) (Strictly typed)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/)
* **Backend / Database:** [Supabase](https://supabase.com/) (PostgreSQL, Authentication)
* **Package Manager:** [Bun](https://bun.sh/)
* **AI Assistance:** Built with the help of [Lovable](https://lovable.dev/)
